# game_carrot
 당근 뽑는 게임  https://kimhyera.github.io/game_carrot
 
### 주요기능
<pre>
1. 지정된 시간안에 모든당근을 클릭해야 이긴다. 

2. 지정된 시간안에 당근을 클릭못하면 게임에서 진다.

3. 버그를 클릭하면 게임에서 진다.

</pre>


### js 순서
<pre>
1. 기본값, 노드변수, 전역변수, 상태변수 를 정의한다.

2. 주요 게임 함수를 만든다. startGame, stopGame, finishGame

3. 주요게임안에 처리해야할 함수를만든다. 

4. 이벤트 함수 만들어서 처리함.
</pre>
